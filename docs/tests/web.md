## formatDuration.test.ts
| Test Case | Type | What is checked |
|-----------|------|-----------------|
| возвращает "—" когда finishedAt = null | Negative | Возврат дефиса, если дата завершения отсутствует. |
| минимум 1 минута даже при нулевой разнице | Boundary | Минимальное отображаемое время составляет 1 минуту, если начало и конец совпадают. |
| округляет до целых минут (вверх при ≥30 сек) | Boundary | Логика округления секунд до минут (менее 60 сек округляется до 1 мин). |
| менее часа → только минуты | Positive | Отображение только минут, если длительность меньше 60 минут. |
| ровно 1 час → "1 ч" | Positive | Форматирование длительности ровно в 1 час без минут. |
| 1 час и несколько минут | Positive | Комбинированное отображение часов и минут. |
| ровно 2 часа | Positive | Корректное отображение нескольких целых часов. |
| несколько часов + минуты | Positive | Комбинированное отображение нескольких часов и минут. |
| очень короткий интервал → всё равно 1 мин | Boundary | Округление интервала в несколько секунд до 1 минуты. |
